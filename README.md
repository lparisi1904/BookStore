# BookStore
Web API Project using .NET 7.0

## Technologies
- .NET 7
- Entity Framework 7
- Fluent API
- AutoMapper
- Swagger
- SQL Server

### Unit tests
- xUnit
- Moq
- Fluent Assertions
- SQLite In-Memory database

## Architecture
- 3 Layers:
  - Application layer (API)
    - Controllers
    - Dtos
  - Domain layer
    - Models
    - Interfaces
    - Services
  - Infrastructure layer
    - Repository Pattern
