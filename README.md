# BookStore
Web API Project using .NET 7.0

## Technologies
- .NET 7
- Entity Framework 7
- Mapster Mapper
- Swagger
- SQL Server

### Unit tests
- xUnit
- Moq
- Fluent Assertions

## Architecture
- 3 Layers:
  - Application layer (API)
    - Controllers
    - Dtos

  - Domain layer
    - Entities
    - Interfaces
    - Services
    
  - Infrastructure layer
    - Repository Pattern
